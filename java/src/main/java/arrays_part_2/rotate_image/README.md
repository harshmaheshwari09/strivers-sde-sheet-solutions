# Rotate Image

This directory contains the boilerplate for the LeetCode problem [Rotate Image](https://leetcode.com/problems/rotate-image/description/).

## Problem Description

You are given an `n x n` 2D matrix representing an image. Rotate the image by 90 degrees clockwise in place.

## Approach

The standard in-place approach is:

1. Transpose the matrix.
2. Reverse each row.

## Time Complexity

O(n^2)

## Space Complexity

O(1)
