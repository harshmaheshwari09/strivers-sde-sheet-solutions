# Find Missing and Repeated Values

## Problem Summary

Given an `n x n` grid containing values from `1` to `n^2`, exactly one value appears twice and exactly one value is missing. Return `[repeated, missing]`.

## Example 1

Input:
```text
grid = [[1,3],[2,2]]
```

Output:
```text
[2,4]
```

## Example 2

Input:
```text
grid = [[9,1,7],[8,9,2],[3,4,6]]
```

Output:
```text
[9,5]
```

## Notes

- The matrix contains numbers from `1` to `n^2`.
- Exactly one number repeats and one number is missing.
