# Two Sum

This directory contains the boilerplate for the LeetCode problem [Two Sum](https://leetcode.com/problems/two-sum/description/).

## Problem Description

Given an array of integers `nums` and an integer `target`, return the indices of the two numbers such that they add up to `target`.

## Approach

The standard approach uses a hash map to store previously seen values and their indices while scanning the array once.

## Time Complexity

O(n)

## Space Complexity

O(n)
